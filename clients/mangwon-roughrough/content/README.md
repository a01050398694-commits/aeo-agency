# 망원 러프러프 — 콘텐츠 디렉토리

여기에 생성되는 콘텐츠:
- `answer-blocks/` — Gemini가 직접 인용할 60~150자 답변 블록 (1쿼리 = 1파일)
- `faq/` — FAQPage Schema.org 마크업용 Q&A
- `blog/` — 장문 가이드 (망원동 카페 큐레이션 등 콘텐츠)
- `social/` — 인스타 캡션, 네이버 블로그 초안
- `schema/` — JSON-LD 마크업 (Organization, FAQPage, LocalBusiness)

## 배포 채널 우선순위 (사이트 없음 가정)
1. **네이버 스마트플레이스** — 사장님 권한 위임 후 소개·메뉴·소식 업데이트
2. **인스타그램 비즈니스** — 정기 포스팅 (시그니처 + 풍경)
3. **네이버 블로그** — 자체 또는 협업 블로거 (FAQ형 후기)
4. **다이닝코드 / 망고플레이트 / 캐치테이블** — 등록 + 메뉴 정확화
5. **Google 비즈니스 프로필** — 등록 후 Q&A 활용
