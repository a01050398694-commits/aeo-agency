## 부속서 A — 측정 방법 명세

### SOV 측정 방법
1. 카테고리 쿼리 30~80개 (사전 합의) 자동 측정
2. Gemini 2.5 Flash + google_search grounding 사용
3. 답변 본문에 클라이언트 브랜드명(또는 도메인) 등장 여부 카운트
4. SOV = (cited 쿼리 수) / (성공 쿼리 수) × 100
5. Rate Limit으로 실패한 쿼리는 분모 제외

### Ahrefs Brand Radar 외부 검증
- 분기 1회 외부 측정 도구로 교차 검증
- ChatGPT / Gemini / Perplexity 다중 LLM 통합 점수

### 매체 게재 정의
- 디에디트 / 플래텀 / 매거진B / 시티호퍼스 등 사전 합의 매체에 갑의 브랜드가 본문에 1회 이상 언급된 경우
- 보도자료 형태는 제외 (자연 큐레이션만 인정)
